CSS Syntax

A CSS rule consists of a selector and a declaration block, where properties are styled.

Elements

  1. Rule set: Composed of a selector and one/many delcarations surrounded by curly braces.
  2. Selector: Identifies which HTML element should be targeted for styling.
  3. Declaration: Assigns a value to a property, separated by a colon. Each declaration must end with a semi-colon.
  4. Property: The type of design you want to add, such as color, width, background color, border, etc.
  5. Value: Accepted value for the assigned property, such as blue, 30px, 50%, etc.

Tags syntax

Element selector:

Select all the tags of the same type.

  • html { background-color: #A68F97; }
  • img { width: 100%; }

Targeting multiple elements:

If you’d like to style more than one element at a time, you can use a comma to separate the tag names.

  • h1, h2, h3 { font-family: sans-serif; }

Class selector (.):

Select an element that has a class.

  • .card { background-color: #F2F2F2; width: 50%; border-radius: 8px; margin-bottom: 1cm; }
  • .card-text { padding: 0 1cm 1cm; }

ID selector (#):

Select an element that has an ID (you can only have one unique ID per page).

  • #section-1 { background-color: mistyrose; }

Descendant selector:

Select an element that's a descendant of another element, which refers to elements nested within other elements. To create a descendant selector, we'll need to use two or more selectors separated by white space.

  • nav a { color: #fff; }
  • ul li { font-size: 1cm; }

Child selector (>):

Select an element directly inside another element.

  • div > h3 { text-transform: uppercase; }

Adjacent sibling (+):

Select an element immediately beside another element.

  • h2 + p { color: orchid; }

General sibling (~):

Select an element that’s at the same level.

  • h3 ~ a { background-color: #000; color: f2f2f2; padding: 0.25 0.75; }

Combine multiple types of selectors:

  • #section-1 strong { color: green; }

Attribute ([]):

Select an element by it’s attribute.

  • a [target="_blank"] { font-weight: bold; color: #f80; }

Pseudo classes

:first-child

Select the element when it’s the first inside its parent.

  • ul li:first-child { background-color: #ff4858; }

:last-child

Select the element when it’s the last inside its parent.

  • ul li:last-child { background-color: #f29947; }

:nth-child ( )

Select an element by it’s number.

  • ul li:nth-child(3) { background-color: #346f48; }

:first-of-type

Select an element that’s the first of its kind within its parent.

  • p:first-of-type { background-color: purple; }
