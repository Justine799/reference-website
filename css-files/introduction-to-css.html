Introduction to CSS

What is CSS?

Also calledCascading Style Sheets, CSS is the language we use to control the appearance of our HTML pages.

How to create a CSS file:

Create a CSS file named style.css in a code editor (such as Visual Studio Code). After naming the file properly (with .css extension), put it within the CSS folder.

How to attach a CSS file:

We need to link the CSS file to the HTML in order for the browser to know where to look for the CSS file.

To do so, attach the CSS file within the (<)head(>) of the HTML document using the (<)link(>) element.

The link element

It's main use is to attach CSS to the HTML file. It contains two important attributes:

  • href: This attribute is used to point to the location of the CSS file (has to point to correct folder).
  • rel: This attribute is used to define this item as a stylesheet.
  • (<)link rel="stylesheet" href="css/style.css(>)
  • This is placed in the (<)head(>)
