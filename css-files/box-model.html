The box model

What is the CSS box model?

The box model is a box that wraps around every HTML element. It's made up of four parts:

  1. Content
  2. Padding
  3. Border
  4. Margin

Content

It's the stuff people see in the browser, the information that is presented to people (the rest is how it is presented). The content defines the basic box.

Padding

Adds spacing inside the box, and pushes the edge of the box away from the content (the background color is visible with the padding). It is often used to give the content some breathing room.

Border

At the edge of the padding is a stroke around the edges of the box called the border. This is where the background color stops.

Margin

Outside the box is the margin, a transparent layer pushing other boxes away. The margin is the space around an element that separates it from other elements. Margins can have positive or negative values.

  • We can specify margin and padding with their individual property, but it's often easier to use the shorthand version.
  • Start at the top and make your way clockwise around the box (top, right, bottom, left).
