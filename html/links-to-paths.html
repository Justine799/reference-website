Links to paths

Link to an absolute URL path

Absolute URLs offer a full address for a website or other resource on the internet.

  • (<)a href="Full link"(>) Link to website (<)/a(>)

Link to a relative URL path

A relative file path points to a file relative to the current page.

  • (<)a href=" Partial link, without www.website.etc"(>) Page in same folder as index.html (<)/a(>)
  • (<)link src="../css/style.css" rel="stylesheet"(>)

Links to elements on the same page

  • (<)a href="#products"(>) Products (<)/a(>)
    • = This is inserted in the main navigation.
  • (<)section id="products"(>)
    • = This is inserted at the start of each section.
